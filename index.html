Brief overview

My name is Omphalus Harkie Kua AKA Ompa. I am a creative technologist that likes to create a contemporary style of art work through Graphic Design, Web Design, Web Development, Moving Images, Music and Visual arts.
I am pretty good with the Adobe Creative Suite which I use for most of my my art work in one way or another. However the creation of this Online Portfolio is a testament of my eagerness to learn to code and become a frontend web developer. Here I focus mainly on layout and structure with consideration for UI and ux.
Ompa's Portfolio is still a mile from the finish line and it shall remain as a working progress to reflect the growth of my skills and knowledge in web. All development stages(commits) for this project have been stored as a repository under the name 'The Big2' on my 'haluskua' Github page.
Ompa's portfolio is a product of my coding and design skills with general consideration for UI/UX. Not all my artwork are included in this portfolio.

Enjoy!

I took a full year course at Yoobee school in 2016 to learn Web development. There, my graphic design skills helped me build or at least design great frontend User Interface with wire-freframes and mockups with the help of Adobe suite.
Yet, with no prior exposure to any sort of coding in html5 and CSS or the javascript programming language, I was insanely petrified when I first saw what code looked like on a black computer screen. However, making a simple Html page come alive was really satisfying. I was never the best student in class but I was certainly keen to learn to find my style in the web development field. I am still learning a whole bunch and I wish to expand further and become a great programmer for Apps one day. This Digital Portfolio you are currently viewing is a result of a persistance in self learning.

Development Setup

I use either Sublime text or Atom as my main IDE. I use gulp to pre-compile my sass, minify script.js, minify index.html, minify css, minify images, gulp-watch and gulp-autoprefixer to handle any flex-box items in my sass. I have been leaning the sass version of css as I find the code a little bit easier to read and execute which I believe is much to do with the fact that I am a designer. I totally enjoy using Flexbox. I believe the two top frameworks which are Bootstrap and Foundation have incorporated the Flexbox classes into their setup. I am familiar with both frameworks but have yet to finish a whole project so it is certainly my next goal. I use gitbash to upload all my "commits" from my local computer up to my github page "haluskua".

What I've Learned

I am fairly confident with graphic design in the scheme of Creativity. I have found a real motive and a passion for it. I admire Soul Bass's work. I am still learning to perfect my style.

Read more

I like to think that any design is a 'navigational path' to or from one's perspective. Therefore a good design allows for a person or persons to connect with it gracefully. It certainly serves as a means of relaying a message or force an action, feeling or thought upon the audience as Graphic Design is mainly visual. Therefore Graphic design creates a visual emotion. Basically, if you can see it, you are connected.
Design elements such as Typography, Shapes, Colors, Texts, Images, Size and composition can alter one's emotions just like how you would feel while listening to music. They are all very important yet I find the composition to be the most detrimental factor in graphic design because composition brings everything together as one like a music track.

Tyography

Using different typefaces helps with my design to focus the theme to a certain genre, whether it be retro, contemporary or a particular theme of the design. It is an important tool to use in graphic design where I use it to improve the final outlook of a design. My two favorite typefaces are Helvetica and Open Sans.

Portfolio

Summary

The code for this portfolio does not include any Frame-works like Bootstrap or foundation. I created my own classes, and id's and with some Jquery and Javascript code along with the Html5 and Css so that I can learn the relationship better. The code is not entirely 'DRY' checked. I still have yet to tidy up repetitive code etc and make the code more legible.

Eventually, I plan on using one of the major frameworks of either Bootstrap or Foundation, to re-build this portfolio on.

I have used gulp to precompile and minify the sass, script.js, and images files and HTML. I have also run autoprefixer to handle the Flex-box items. I have tried to make the site Responsive across mobile, tablet and desktop. I have not tested it on a *tablet yet. Most of the device testing has been done on my phone and using Chrome's inspector tool for device testing. There are minor bugs on a mobile device in Landscape view mainly with the drop-down menu not showing after automatically switching views from Portrait back to landscape.
As for browser compatibility, I have not done any resolution testing on any other browser apart from Chrome. As mentioned earlier, the site is a working progress. Therefore I will work on each bug from time to time to get the app working fluidly through all browser.

Ompa comes from the beautiful Chimbu Province situated in the heart of Papua New Guinea. He arrived in New Zealand in 1999 on an academic scholarship. He has since learned to live a simple and wonderful life here in beautiful Aotearoa.
